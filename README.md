### コミットメッセージの規則
コミットメッセージの接頭辞には，以下のどれかを付ける．
| 接頭辞 | 内容 | 
| :---   | :--- | 
| `feat` | 既存のファイルに新しいコンテンツを追加する |
| `fix`  | 既存のファイルの一部を修正する |
| `add`  | 新しいファイルを追加する |
| `docs` | `README.md`などのwebページに関係ない`*.md`を変更する |
